合宿1日目：まとめ

○git用フォルダの作成

ターミナルを起動し、 mkdir フォルダ名 を実行。

○gitの設定

git init →初期設定のコマンド
git config --global user.email"自分のメアド" →メールアドレスの設定
git config --global user.name"名前（アルファベット） →ユーザー名の設定

○GitHubの使い方

・リポジトリの作成

右上の＋のボタンをクリックし、 「New repository」 を選択。
↓
「Repository name」 に名前を入力。
「Public」 と 「Add a README file」 を選択し、 「Create repository」 をクリック。
↓
リポジトリ完成

・アクセストークンの作成

自分のアイコンをクリックし、 「Setting」 を選択
↓
「Developper setting」 をクリック
↓
「Personal access tokens」 をクリック
↓
「generate new token」 をクリックし、パスワードを入力
※アクセストークンはコピペしておく。

・リポジトリーをクローンする

cd mygit のコマンドを実行し、mygit（専用に作ったフォルダ）に移動。
↓
git clone リポジトリのURL を実行する。
↓
repository nameで設定した名前のフォルダができているか確認。あと、そのフォルダの中の「README.md」も

・公開（ファイルの追加、変更の時も）

次のコマンドを順に実行
cd フォルダ名/リポジトリー名
git add .
git commit -m"メッセージ"
git push
